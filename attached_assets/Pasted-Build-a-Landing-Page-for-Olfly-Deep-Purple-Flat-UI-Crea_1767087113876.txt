Build a Landing Page for Olfly (Deep Purple Flat UI)

Create a responsive marketing landing page for an app called Olfly. The app helps people regain their sense of smell using guided olfactory training routines. Build this as a single page site using React + Vite + TailwindCSS. Match the app’s deep purple flat UI style.

Brand and style

Brand name: Olfly

Tagline: “Wake up your super sniffer.”

Visual style: modern, flat, clean, deep purple theme

Colors:

Background: #1A0F35

Card: #2B215B

Accent gradient: #DF37FF to #A259FF

Text: white and #B9AEE2

Typography: modern sans serif, bold headers, friendly tone

Icon style: thin outline icons

No wave textures. Minimal gradients only.
